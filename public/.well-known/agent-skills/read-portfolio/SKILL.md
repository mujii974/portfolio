# Read portfolio

This skill describes how an AI agent can read and summarize the
mujii.dev portfolio. There is no interactive tool or API call involved —
this is a guide to the site's static, publicly readable content.

## What this site is

A personal portfolio for Mujtaba Shahid, a Data & Cybersecurity
undergraduate (UDST, Doha) focused on penetration testing, zero-trust
architecture, and secure software development.

## How to find information

1. Fetch `https://mujii.dev/llms.txt` for a short summary (sections, key
   projects, contact links).
2. Fetch `https://mujii.dev/llms-full.txt` for full context: about,
   experience, education, certifications, projects, skills, and contact
   details.
3. For project source code, follow the GitHub links listed in
   `llms-full.txt` (all under https://github.com/mujii974/).
4. For the CV, fetch `https://mujii.dev/cv/Mujtaba_Shahid_CV.pdf`.

## What this skill does NOT do

- It does not expose any interactive action, form submission, or write
  capability.
- It does not represent a real API, OAuth flow, or MCP server — none
  exist on this site. See `/auth.md`.
- It should not be used to contact Mujtaba on his behalf without the
  requesting user's explicit intent; the contact channels are for humans
  reviewing his work.
